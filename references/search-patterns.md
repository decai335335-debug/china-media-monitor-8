# 各媒体搜索模板（题材无关通用版）

## 搜索策略说明

每家媒体对每个目标题材执行 **3 轮搜索**，由宽到精：

| 轮次 | 目的 | 组合公式 |
|------|------|----------|
| 第1轮 | 宽泛抓取 | `[媒体名] [题材核心词] 年份` |
| 第2轮 | 精准锁定案例 | `[媒体名] [题材子题词] 案例/实践/落地` |
| 第3轮 | 补充/站内检索 | `site:[域名] [题材子题词]` |

> 题材关键词来自 `references/media-matrix.md` 的主题词库。
> 示例替换：`[题材核心词]` → `基层治理` / `城市更新` / `外来物种` 等。

---

## 通用搜索模板（适用于所有 25 家媒体）

```
搜索词组（T = 目标题材）：
1. [媒体名] [T核心词] 案例 2026
2. [媒体名] [T子题词] 落地 / 实践 / 试点
3. site:[媒体域名] [T核心词]
4. [媒体名] [T子题词] 调查 / 报道
```

## 备用搜索策略（当 site: 搜索无结果时）

```
策略 1：聚合平台搜索
- "[媒体名] [T核心词] 案例" site:weixin.qq.com
- "[媒体名] [T子题词]" site:mp.weixin.qq.com

策略 2：百度新闻搜索
- [媒体名] [T核心词] 案例 -评论 -理论 -社论

策略 3：微信公众号直搜
- 直接搜索媒体官方公众号名称 + 题材关键词

策略 4：题材反向检索
- 当媒体名限定过严时，先按 "[T核心词] 案例 [年份]" 搜索，再筛 25 家媒体域名
```

## 搜索词长度控制

- 每次查询 **4~10 个字** 为宜
- 避免一次堆叠 3 个以上题材词
- 若有多题材任务，**按题材分别搜索**，不要合并
